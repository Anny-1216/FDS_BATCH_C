# FDS_BATCH_C

## Description
FDS_BATCH_C is a collection of C programs focusing on various data structures and algorithms. The following programs are included:

1. String Manipulation
2. Selection Sort
3. Bubble Sort
4. Linked List
5. Stack
6. Circular Queue
